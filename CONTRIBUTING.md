# Contributing

Thanks for your interest in contributing.

## What we accept
- UI/UX improvements
- Data model enhancements (schemas, validation)
- Accessibility improvements (a11y)
- Documentation and source transparency improvements

## What we do NOT accept
- Harassment, hate speech, or calls for violence
- Unsourced or misleading claims

## Development
1. `npm install`
2. Create `.env.local` with `GEMINI_API_KEY=...`
3. `npm run dev`

## Pull Requests
- Keep PRs small and focused
- Add/Update citations for any new metrics
